# My Best IDE Font

[JetBrains Mono](https://www.jetbrains.com/lp/mono/) — a typeface designed specifically for developers.

It has a higher letter height, reduced noise from unnecessary details, and ligatures for common code symbols (`->`, `!=`, `===`, etc.). Available for free in any IDE.
