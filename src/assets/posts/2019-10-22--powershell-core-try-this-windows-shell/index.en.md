# PowerShell Core — Try This Windows Shell

If you're seeking an enhanced command-line experience on Windows, try PowerShell Core. It looks great when paired with modern terminal applications.

## Recommended Terminal Options

- [Windows Terminal](https://aka.ms/terminal) — Microsoft's official terminal application
- [Terminus](https://github.com/Eugeny/tabby) — A cross-platform terminal
- [Fluent Terminal](https://github.com/felixse/FluentTerminal) — A UWP-based terminal

Download PowerShell Core from the [GitHub releases page](https://github.com/PowerShell/PowerShell/releases).
