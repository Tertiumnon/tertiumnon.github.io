# Your Categories

How to organize your bookmarks? Here are a few approaches:

**Theme → Names**

```
Media / Bloomberg
Media / New York Post
```

**Type → Theme → Names**

```
Apps / Office / Libre Office
Apps / Office / Microsoft Office
```

**Type → Theme → Type → Names**

```
Apps / Development / IDE / Visual Studio Code
Apps / Development / IDE / PHPStorm
```

**Content type → Content type → Theme**

```
Audio / Music / ...
Video / Movies / Action /
Video / Movies / Comedy /
```
