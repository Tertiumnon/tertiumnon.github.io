# Send Text To Form (Chrome Extension)

A Chrome browser extension for sending selected text from web pages to a custom form destination.

Highlight any text on a webpage and use the context menu "Send to…" option to route it to a pre-configured web form. Useful for SEO analyzers and other text processing tools.

[Send Text to Form on Chrome Web Store](https://chrome.google.com/webstore/search/send%20text%20to%20form)
