# How to Organize Your Wallpapers

Create a folder "Wallpapers" and then create theme-subfolders like "nature", "space", "movies/Game of Thrones", etc.

Then download [Advanced Renamer](https://www.advancedrenamer.com/) and use this pattern in the "New Name" field:

```
<Img Width>x<Img Height> - <DirName:1> - <Inc NrDir:1>
```

Add your "Wallpapers" folder for batching and click "Start batch".

Your files will be renamed to something like `1920x1080 - nature - 001.jpg`, making them easy to sort and identify.
