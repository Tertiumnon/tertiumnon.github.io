# Oh My Zsh Analogue for PowerShell

After years of searching for a Windows equivalent to Oh My Zsh, I found a solution based on Scott Hanselman's blog post: "How to make a pretty prompt in Windows Terminal with Powerline, Nerd Fonts, Cascadia Code, WSL, and oh-my-posh."

The solution is [oh-my-posh](https://ohmyposh.dev/) — it enables customization of PowerShell prompts with Powerline fonts and Nerd Fonts integration.

The only thing I still miss is Oh My Zsh history suggestions. But overall it's a solid upgrade to the default PowerShell prompt.
